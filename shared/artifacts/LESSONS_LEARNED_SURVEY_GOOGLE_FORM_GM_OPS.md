# TGIF Phase 2 Rollout — Lessons Learned (GM / Ops Survey)

## Form Title
TGIF Phase 2 Rollout — Lessons Learned (GM / Ops Survey)

## Form Description
Purpose: Capture specific, actionable learnings from Phase 2 go-lives so we can improve the remaining rollout waves.

Please answer based on what actually happened at your store(s). This will be used to update the playbook, FAQs/best practices, training, and support/issue management.

---

## Section 1 — Respondent + Store Context (Required)

1) Name
- Type: Short answer

2) Role
- Type: Multiple choice
- Options:
  - GM

  - Ops Leader
  - Other (short answer)

3) Franchise group
- Type: Short answer

4) Store / Location
- Type: Short answer

5) Go-live date (approximate is fine)
- Type: Date (or short answer)

6) Were you on-site for go-live?
- Type: Multiple choice
- Options:
  - Yes
  - No

9) Stability timeline
- Type: Multiple choice

  - Stable in < 3 days
